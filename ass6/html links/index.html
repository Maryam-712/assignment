How links can be useful

Absolute URLs

When you link to a different website, the value of the href attribute will be the full web address for the site, which is known as an absolute URL

example

https://www.google.com

Relative URLs

When you are linking to a page on your own website, you do not need to specify the domain name. You can use relative URL

example

style.css

Email link

To create a link that starts up the user's email program and addresses an email to a specified email address

example

msmaryammansoor712@gmail.com

Linking to the same page

At the top of a long page you might want to add a list of contents that links to the corresponding sections lower down. Or you might want to add a link from part way down the page back to the top of it to save users from having to scroll back to the top.
